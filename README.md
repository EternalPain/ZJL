# ZJL 免流防跳脚本
****

### 注意事项:
    1. tiny模式的uid写0 3003 3004之一 不然会没网
    2. 本机成功代理了UDP的话 共享UDP放行将无效
    3. 如果开了免流后手机有什么奇怪的毛病的话 试试防跳放行IPv6

### 选项（不带参数）:
    -o 或 --open                   开启
    -c 或 --close                  关闭
    -d 或 --display                显示界面
    -i 或 --info                   显示信息
    -h 或 --help                   查看帮助
    -v 或 --version                查看版本

### 更新日志:
     1. 把 [开关数据] 功能增加回来 并更智能
     2. 修复同一WiFi下 本地视频不能投屏
     3. 修复不能获取安卓11的WiFi密码、热点的名字和密码
